# Notification Feature Requirements

## Overview
알람(Notification) 기능 추가를 위한 요구사항 문서

---

## Requirements

### 1. 기능 
- MCP에 상태 변화가 일어났을 때 (댓글, 즐겨찾기, status가 pending에서 approved로 변경 등) Navbar username 쪽에 알람 표시 추가

### 2. 알람 트리거 조건
- 내가 등록한 MCP에 댓글이 달리거나 즐겨찾기가 추가되었을 경우
- 등록한 MCP가 관리자의 승인을 받아 pending에서 approved로 변경되었을 경우
- 관리자인 경우에, 새로 등록된 MCP가 있을 경우

### 3. 알람 UI/UX
- Navbar Username 쪽에 버튼을 누르면 notification 추가 (facebook, instagram과 같은 형식)
